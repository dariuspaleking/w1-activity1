About Web Design

Web design is the process of creating websites. It includes both the visual appearance and the functionality of a website.

A webpage is a single document on the internet, while a website is a collection of related webpages grouped under a single domain name.

Key Web Design Concepts

HTML

HyperText Markup Language - the structure of web pages

CSS

Cascading Style Sheets - the styling of web pages

JavaScript

JavaScript is a programming language that makes web pages interactive
